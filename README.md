 Movie App

Приложение для просмотра популярных фильмов с возможностью добавления в избранное.

Stack: Kotlin, Jetpack Compose, MVVM, Room, Volley, Coroutines  

Архитектура:
UI → ViewModel → DAO → Room  

- Загрузка фильмов с TMDB API  
- Работа с избранным (Room)  
- Парсинг JSON  
- Управление состоянием через Compose
